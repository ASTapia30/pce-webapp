# Proyecto de Estimación de Costos de Proyectos

Este proyecto es una aplicación web para la gestión y estimación de costos de proyectos, enfocándose en la creación y administración de trabajadores. La aplicación permite agregar, editar y visualizar información sobre los trabajadores, incluyendo su nombre, correo, salario, profesión, ubicación y categoría de trabajo.

## Estructura del Proyecto

El proyecto tiene la siguiente estructura de archivos:

```
pce-webapp
├── src
│   ├── index.html          # Página principal de la aplicación
│   ├── views
│   │   └── workers.html    # Página para gestionar trabajadores
│   ├── css
│   │   └── styles.css      # Estilos CSS de la aplicación
│   └── js
│       ├── workers.js      # Lógica JavaScript para manejar trabajadores
│       └── data.js         # Simulación de datos de trabajadores
├── package.json            # Configuración de npm
└── README.md               # Documentación del proyecto
```

## Instalación

1. Clona el repositorio en tu máquina local.
2. Navega al directorio del proyecto.
3. Ejecuta `npm install` para instalar las dependencias necesarias.

## Uso

1. Abre `src/index.html` en tu navegador para acceder a la aplicación.
2. Utiliza la página de gestión de trabajadores para agregar nuevos trabajadores, editar información existente y aplicar filtros de búsqueda.
